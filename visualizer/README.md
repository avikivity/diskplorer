# Diskplorer visualizer
Web based visualizer for fio/diskplorer results. You can use either diskplorer csv files or fio json results file.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files

